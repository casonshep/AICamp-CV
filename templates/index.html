Turtles

A short project designed to detect a few species of turtle.

Why is this project important? Try it out!

Built with Bootstrap 5, Flask, and a number of amazing libraries

Turtle Classification, yes. is my project for the Computer Vision section of the AI Camp crash course curriculum! This project was intended for use identifying species of turtles that users encounter wherever they may be. This project uses a machine learning model to classify images of turtles as either a Loggerhead Sea Turtle, Red-Eared Slider, Bog Turtle, or Snapping Turtle. The model was trained on 400 images. The model is a simple convolutional neural network, and was trained using the Transformers library. The web app was built using Flask, and the front end was built using Bootstrap 5. The model was trained on Google Colab, and the web app is hosted through replit.

I started by using a modified script from SerpAPI to scrape the dataset from Google Images. I then used Roboflow to label the images into the four turtle categories: Loggerhead, Bog, Red-Eared Slider, and Snapping. At this point it was relatively simple to download the images through the Roboflow API, and process them in a Google Colab notebook with the Transformers library. The trained model was then uploaded to HuggingFace, and the web app was built using Flask, using the model and HuggingFace's inference APIs to make calls to the model.

Why is this important?

Currently, over half of the ocean's species of seas turtle are classified as endangered. These are due to the actions of human's, and our poor care for our oceans.

Species Tracking

In addition to taking care of turtles, it is crucial to monitor different turtle species for several reasons. Firstly, monitoring helps us understand the population dynamics, distribution, and habitat requirements of each species. This knowledge allows us to implement effective conservation strategies tailored to the specific needs of different turtle populations. Secondly, monitoring helps identify any declines or changes in species abundance, which can be early indicators of environmental disturbances or threats. Lastly, monitoring different species provides valuable scientific data that contributes to broader research efforts and our understanding of ecosystem health and functioning. Through comprehensive monitoring programs, we can ensure the long-term survival and well-being of turtles and the ecosystems they inhabit.
